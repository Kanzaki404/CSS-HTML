robot.txt

published date: 19 / 09 / 2018

Robots.txt is a file located in the root folder that contain directives for the search engines. This files can prevent search engines reaching specific parts of the website. There are few things to keep in mind when creating robots.txt files to the website:

  1. The file has the potential to make big parts of the website inaccessible to the search engines.
  2. robots.txt should be set in the root of the website.
  3. The file is only valid for the full domain it resides on including protocol such as http/https
  4. Avoid the usage of crawl-delay directive for search engines.
